## v1.0.4 — 원문/번역문 편집 & 원문 보호 강화

### ✨ 새 기능

**원문/번역문 편집 분리**
번역된 메시지 이름 옆에 🐟(고양이)/🍖(호랑이) 편집 버튼이 추가됩니다.
- ST 연필 ✏️ → 원문(영어) 편집
- 🐟/🍖 → 번역문(한국어) 편집
- 번역문 수정 후 저장해도 영어 원문이 자동 보존됩니다
- 편집 취소 시 번역문이 자동으로 복원됩니다

### 🔧 버그 수정

**원문 오염 자동복구**
ST 내부 렌더링 과정에서 번역문이 원문을 덮어쓰는 현상에 대한 자동 복구 워치독을 추가했습니다. 로드 시, 채팅 전환 시, 10초 간격으로 상시 감시하여 오염 감지 즉시 복구합니다.

**에러 알림 한국어화**
API 오류 메시지가 한국어로 표시됩니다.
- 🔑 API 키가 유효하지 않아요 (401)
- 🚫 API 접근 권한이 없어요 (403)
- ⏳ 요청이 너무 많아요 (429)
- 💥 서버 오류가 발생했어요 (500)

**벌크 이모지 순서 수정**
🦁20개 → 🐯15개 → 🐱10개 → 🐭5개 순서가 올바르게 표시됩니다.

## v1.0.3 — 번역 품질 대폭 개선 & 설정 안정화

### 🔧 버그 수정

**설정 초기화 버그 수정**
채팅 전환 시 스타일/온도/추가지시사항이 초기화되는 버그를 수정했습니다. 전역 기본 설정이 영구 보존되며, 프리셋이 기본 설정을 오염시키지 않습니다.

**사전 오적용 방지**
사전에 등록된 단어가 관련 없는 캐릭터에 잘못 적용되는 현상을 수정했습니다. 본문에 실제 존재하는 단어만 프롬프트에 전송합니다.

### ✨ 새 기능

**번역 스타일 4종 추가**
기존 3종(일반/소설/캐주얼)에 4종이 추가되었습니다.
- 번역체 탈피 — 자연스러운 한국어 어순
- 존댓말 고정 — 해요체 구어
- 반말 고정 — 친구 말투
- 문어체 — 출판 소설풍

**벌크 번역 병렬화**
동시 2개 메시지를 병렬 처리하여 벌크 번역 속도가 약 2배 향상되었습니다.

**사전 이중 안전망**
API 호출 전 고유명사를 자동 치환하고, 프롬프트에도 Glossary를 동시 전달하여 사전 적용률이 크게 향상되었습니다.

### 📈 번역 품질 개선

**systemInstruction 분리**
Gemini API의 systemInstruction 필드를 활용하여 시스템 프롬프트가 번역 대상으로 오인되는 현상을 방지합니다.

**CONTENT FIDELITY 규칙**
- 원문에 없는 내용 추가 금지 (ZERO ADDITION)
- 문장 빼먹기/요약 금지 (ZERO OMISSION)
- 감정 강도 100% 보존 (EMOTION INTENSITY)
- 물리 동작 구체성 유지 (ACTION PRECISION)
- 캐릭터 말투 일관성 유지 (SPEECH CONSISTENCY)

**스타일별 few-shot 예시**
각 번역 스타일에 실제 번역 전/후 예시가 내장되어 번역 정확도가 향상되었습니다.

**문맥 화자 정보**
이전 메시지에 화자 이름(캐릭터/유저)을 포함하여 캐릭터별 말투 일관성이 향상되었습니다.

## v1.0.1 — 버그 수정 & 스마트 감지 강화

### 🔧 버그 수정

**스와이프 번역 덮어쓰기 수정**
스와이프(< >)로 다른 대사를 선택한 뒤 번역하면 이전 대사의 번역이 나오던 문제를 수정했습니다.

**수정창 캐시 꼬임 수정**
수정 모드에서 텍스트를 변경한 뒤 번역하면 변경 전 문장의 캐시가 나오던 문제를 수정했습니다.

**사전 단어 원문 병기 방지**
사전에 `Soap=소프`를 등록하면 번역 결과가 `소프[Soap]`처럼 원문이 함께 출력되던 현상을 수정했습니다.

### ✨ 새 기능

**일본어 · 중국어 스마트 감지**
기존 한국어 ↔ 영어 자동 감지에 더해, 일본어와 중국어 텍스트도 자동으로 감지하여 설정한 목표 언어로 번역합니다.

### 💡 사전 활용 팁: 동일 단어 다중 등록

같은 단어에 여러 뜻을 등록할 수 있습니다! AI가 문맥을 읽고 적절한 번역을 자동 선택합니다.

```
Soap=소프
Soap=비누
```

→ "Soap handed the knife" → **소프**가 칼을 건넸다 (캐릭터명)
→ "wash with soap" → **비누**로 씻다 (물건)
